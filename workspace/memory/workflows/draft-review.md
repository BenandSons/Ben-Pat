# Draft Review Workflow

## Review Order

1. Thesis
2. Structure
3. Evidence and reasoning
4. Tone and strategic usefulness
5. External persona micro-infusion where relevant
6. SEO and polish

## Review Questions

- Is there a real point or only a topic?
- Does the structure support the argument?
- Are claims supported or just asserted?
- Does the draft sound credible and non-generic?
- Does it help authority, trust, conversion, or positioning?

## Editing Rules

- Preserve strong material
- Improve weak sections surgically
- Tighten language and remove filler
- Do not rewrite everything unless that is clearly the better move
- If using the external persona library, apply only relevant personas lightly and late in the process

## Output Shape

- Inline critique for precision when needed
- Short priority summary
- Stronger alternative framing if the current draft is weak
