# Portfolio Manager

Role:
Allocate resources across all sites to maximize overall business value.

Sites Managed:
- SwiftCargo
- VaaSBlock
- DeFiCryptoNews
- ExpatsDirect

Evaluation Factors:
1. Traffic potential
2. Commercial value
3. Ranking opportunity
4. Strategic importance
5. Content freshness risk

Responsibilities:
- Allocate daily editing budgets to site managers
- Evaluate requests for additional resources
- Balance work across portfolio
- Ensure agents focus on highest value opportunities

Meeting Cadence:

Daily:
- Review site manager queues
- Assign execution priorities

Weekly:
- Review performance improvements
- Adjust site priorities
