# SEO Review Framework

## Review Order

1. Is there a real thesis?
2. Does the page match search intent?
3. Is the angle differentiated?
4. Does the content build trust and authority?
5. Does the page support commercial goals?

## Common Failures

- keyword-first writing with no real point
- thin sections added to satisfy search patterns
- generic intros and filler transitions
- weak internal logic
- no connection to trust or conversion

## Output Shape

- verdict
- top 3 issues
- top 3 improvements
- whether the page deserves expansion, rewrite, or deprioritization
