# Long-Form Authority Articles

## Purpose
This guide governs long-form authority articles written for company sites, guest editorials, and third-party publications where the goal is trust, discoverability, citation value, and durable authority.

## Scope
This file is only for:
- long-form article goals
- host-site and audience context
- authority-article planning workflow
- competitor and SERP benchmarking
- metadata and schema preparation for articles

This file is not for:
- crypto-specific query psychology
- claim defensibility rules
- product-policy page structure
- edit-pass reconciliation against source files

## Core Objective
Authority articles should:
- answer real search intent
- help real people solve real problems
- be strong enough to be cited or summarized by LLM systems
- build authority for the publishing site and the referenced brand
- contribute something useful to the wider conversation

## Working Standard
- Write for professional editorial quality, not SEO filler.
- The article must have a real thesis, not just a topic.
- The angle should shape the whole piece, not sit on top of it.
- If the idea is weak, generic, or commercially misaligned, say so early.

## Collaboration Rules
- Clarify thesis, audience, host-site context, backlink goals, and commercial sensitivity before drafting.
- Ask sharp questions if they materially improve the article.
- Prefer candid collaboration over polite agreement with a weak brief.

## Planning Workflow
1. Understand the brief
2. Identify search intent
3. Review competing content
4. Research keyword and question behavior
5. Define the angle
6. Build the outline
7. Identify statistics and sources
8. Draft
9. Expand FAQs where warranted
10. Prepare metadata and schema
11. Run editorial tightening

## Competitor Review
When benchmarking content competitors, check:
- what ranks now
- what they all cover
- what they repeat lazily
- what they fail to explain well
- whether the topic is realistically winnable
- what better thesis, framework, checklist, or narrative angle could outperform them

## Question Research
Treat search behavior as a map of what people actually want answered.

Look for:
- standard keyword phrases
- People Also Ask style questions
- conversational LLM-style questions
- long-tail problem framing
- comparison queries
- “what should I know before…” behavior

## Metadata And Schema
Prepare for each article:
- SEO title
- SEO meta description
- X title and description
- OpenGraph title and description
- relevant schema where appropriate, such as Article, FAQPage, Opinion, or Author-related schema

Metadata should reflect search intent clearly first and branding second.

## LLM Visibility
Authority articles should be easy to quote and summarize.

Prefer:
- clear definitions
- concise answer blocks
- frameworks
- examples
- comparisons
- key takeaways near the top when useful
- FAQ sections that extend the article rather than pad it

## What Adds Alpha
- a clear thesis with teeth
- realistic competition assessment
- a more useful angle than what already ranks
- a piece that helps both readers and machine retrieval without reading like SEO
